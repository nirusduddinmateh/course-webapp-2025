## Step 9 ‚Äì GROUP BY / HAVING

`GROUP BY` ‡πÉ‡∏ä‡πâ **‡∏à‡∏±‡∏î‡∏Å‡∏•‡∏∏‡πà‡∏°‡πÅ‡∏ñ‡∏ß** ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡∏™‡∏£‡∏∏‡∏õ‡∏ï‡πà‡∏≠‡∏Å‡∏•‡∏∏‡πà‡∏° (‡πÄ‡∏ä‡πà‡∏ô ‡∏ô‡∏±‡∏ö/‡∏£‡∏ß‡∏°/‡πÄ‡∏â‡∏•‡∏µ‡πà‡∏¢)  
`HAVING` ‡πÉ‡∏ä‡πâ **‡∏Å‡∏£‡∏≠‡∏á‡∏ú‡∏•‡∏•‡∏±‡∏û‡∏ò‡πå‡∏´‡∏•‡∏±‡∏á‡∏à‡∏±‡∏î‡∏Å‡∏•‡∏∏‡πà‡∏°‡πÅ‡∏•‡πâ‡∏ß** (‡∏ï‡πà‡∏≤‡∏á‡∏à‡∏≤‡∏Å `WHERE` ‡∏ó‡∏µ‡πà‡∏Å‡∏£‡∏≠‡∏á‡∏Å‡πà‡∏≠‡∏ô‡∏à‡∏±‡∏î‡∏Å‡∏•‡∏∏‡πà‡∏°)

---

### ‡∏ï‡∏±‡∏ß‡∏≠‡∏¢‡πà‡∏≤‡∏á‡∏û‡∏∑‡πâ‡∏ô‡∏ê‡∏≤‡∏ô

#### ‡∏Ñ‡∏ß‡∏≤‡∏°‡∏ï‡πà‡∏≤‡∏á‡∏£‡∏∞‡∏´‡∏ß‡πà‡∏≤‡∏á WHERE vs HAVING
```sql

```

#### ‡∏à‡∏±‡∏î‡∏Å‡∏•‡∏∏‡πà‡∏°‡∏´‡∏•‡∏≤‡∏¢‡∏Ñ‡∏≠‡∏•‡∏±‡∏°‡∏ô‡πå
```sql
-- ‡∏ô‡∏±‡∏ö‡∏≠‡∏≠‡πÄ‡∏î‡∏≠‡∏£‡πå‡∏ï‡∏≤‡∏° ‡∏•‡∏π‡∏Å‡∏Ñ‡πâ‡∏≤ + ‡∏™‡∏ñ‡∏≤‡∏ô‡∏∞
```

#### ‡∏™‡∏£‡∏∏‡∏õ‡∏¢‡∏≠‡∏î‡∏Ç‡∏≤‡∏¢‡∏ï‡πà‡∏≠‡∏™‡∏¥‡∏ô‡∏Ñ‡πâ‡∏≤ (‡∏à‡∏≤‡∏Å order_items)

## Step 10 ‚Äì JOIN (‡∏î‡∏∂‡∏á‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏à‡∏≤‡∏Å‡∏´‡∏•‡∏≤‡∏¢‡∏ï‡∏≤‡∏£‡∏≤‡∏á)

## 1) INNER JOIN
‡∏î‡∏∂‡∏á‡πÄ‡∏â‡∏û‡∏≤‡∏∞‡πÅ‡∏ñ‡∏ß‡∏ó‡∏µ‡πà **‡∏à‡∏±‡∏ö‡∏Ñ‡∏π‡πà‡πÑ‡∏î‡πâ‡∏ó‡∏±‡πâ‡∏á‡∏™‡∏≠‡∏á‡∏ù‡∏±‡πà‡∏á**

## 2) LEFT JOIN
‡∏î‡∏∂‡∏á ‡∏ó‡∏∏‡∏Å‡πÅ‡∏ñ‡∏ß‡∏ù‡∏±‡πà‡∏á‡∏ã‡πâ‡∏≤‡∏¢ ‡πÅ‡∏°‡πâ‡∏à‡∏∞‡πÑ‡∏°‡πà‡∏°‡∏µ‡∏Ñ‡∏π‡πà (‡∏Ñ‡∏≠‡∏•‡∏±‡∏°‡∏ô‡πå‡∏ù‡∏±‡πà‡∏á‡∏Ç‡∏ß‡∏≤‡∏à‡∏∞‡πÄ‡∏õ‡πá‡∏ô `NULL`)

```sql
  ON c.id = o.customer_id;
```
> ‡∏™‡∏£‡∏∏‡∏õ‡∏™‡∏±‡πâ‡∏ô INNER <br>JOIN ‚Üí ‡πÄ‡∏≠‡∏≤‡πÄ‡∏â‡∏û‡∏≤‡∏∞‡∏ó‡∏µ‡πà‡∏à‡∏±‡∏ö‡∏Ñ‡∏π‡πà‡πÑ‡∏î‡πâ <br>LEFT JOIN ‚Üí ‡πÄ‡∏≠‡∏≤‡∏ù‡∏±‡πà‡∏á‡∏ã‡πâ‡∏≤‡∏¢‡∏ó‡∏±‡πâ‡∏á‡∏´‡∏°‡∏î + ‡∏Ñ‡∏π‡πà‡∏ó‡∏µ‡πà‡∏°‡∏µ (‡∏ñ‡πâ‡∏≤‡πÑ‡∏°‡πà‡∏°‡∏µ ‡∏Ñ‡∏π‡πà‡πÄ‡∏õ‡πá‡∏ô NULL)

## 3) RIGHT / FULL OUTER JOIN (‡πÄ‡∏Å‡∏£‡πá‡∏î)
- **RIGHT JOIN**: ‡∏ï‡∏£‡∏á‡∏Ç‡πâ‡∏≤‡∏° LEFT (‡πÄ‡∏≠‡∏≤‡∏ù‡∏±‡πà‡∏á‡∏Ç‡∏ß‡∏≤‡∏ó‡∏∏‡∏Å‡πÅ‡∏ñ‡∏ß) ‚Äì ‡∏°‡∏µ‡πÉ‡∏ô MySQL/PG
- **FULL OUTER JOIN**: ‡πÄ‡∏≠‡∏≤‡∏ó‡∏±‡πâ‡∏á‡∏™‡∏≠‡∏á‡∏ù‡∏±‡πà‡∏á‡πÅ‡∏°‡πâ‡πÑ‡∏°‡πà‡πÅ‡∏°‡∏ï‡∏ä‡πå ‚Äì MySQL ‡πÑ‡∏°‡πà‡∏°‡∏µ‡πÇ‡∏î‡∏¢‡∏ï‡∏£‡∏á ‡∏ó‡∏≥‡πÑ‡∏î‡πâ‡∏î‡πâ‡∏ß‡∏¢ `LEFT JOIN ... UNION ... RIGHT JOIN ... WHERE ... IS NULL`

## 4) JOIN ‡∏´‡∏•‡∏≤‡∏¢‡∏ï‡∏≤‡∏£‡∏≤‡∏á (‡∏ï‡∏±‡∏ß‡∏≠‡∏¢‡πà‡∏≤‡∏á Order System)
‡∏™‡∏°‡∏°‡∏∏‡∏ï‡∏¥‡∏°‡∏µ `order_items(order_id, product_id, price, quantity)` ‡πÅ‡∏•‡∏∞ `products(id, name)`
```sql
SELECT o.id AS order_id,
       c.name AS customer_name,
       p.name AS product_name,
       oi.quantity,
       oi.price,
       (oi.quantity * oi.price) AS line_total
FROM orders o
JOIN customers c   ON c.id = o.customer_id
JOIN order_items oi ON oi.order_id = o.id
JOIN products p     ON p.id = oi.product_id
ORDER BY o.id;
```

## 5) JOIN + GROUP BY (‡∏™‡∏£‡∏∏‡∏õ‡∏à‡∏≥‡∏ô‡∏ß‡∏ô‡∏≠‡∏≠‡πÄ‡∏î‡∏≠‡∏£‡πå‡∏ï‡πà‡∏≠‡∏•‡∏π‡∏Å‡∏Ñ‡πâ‡∏≤)
```sql
SELECT c.id, c.name, COUNT(o.id) AS total_orders
FROM customers c
LEFT JOIN orders o ON o.customer_id = c.id
GROUP BY c.id, c.name
HAVING COUNT(o.id) > 5;
```

```sql
-- ‡∏•‡∏π‡∏Å‡∏Ñ‡πâ‡∏≤‡∏ó‡∏µ‡πà‡πÑ‡∏°‡πà‡πÄ‡∏Ñ‡∏¢‡∏°‡∏µ‡∏≠‡∏≠‡πÄ‡∏î‡∏≠‡∏£‡πå‡πÄ‡∏•‡∏¢
```
